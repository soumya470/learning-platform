# Full Feature Online Learning Platform

Included Features:
- Homepage with featured courses
- Course listing page
- Course detail cards
- Enroll system
- My Courses dashboard
- Progress tracking
- Lesson completion simulation
- Responsive design
- LocalStorage persistence
- Mock login/signup
- Certificate generation
- Admin panel
- Video player integration
- Search courses
- Filter functionality

Run:
mvn spring-boot:run
Open:
http://localhost:9090
